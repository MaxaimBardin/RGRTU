Viewing PDF Documentation

What You Need

Some of the documentation in this package is in the form of Adobe Acrobat® files. Adobe Acrobat files are stored in a format called Portable Document Format, or PDF for short. To view PDF files, you will need to download and install the Adobe Acrobat Reader from the Adobe Website, if it is not already installed on your computer.

To download the latest Adobe Acrobat Reader please visit: http://www.adobe.com/products/acrobat/readstep2.html

Additional information about the Adobe Portable Document Format is available at: http://www.adobe.com/products/acrobat/adobepdf.html
